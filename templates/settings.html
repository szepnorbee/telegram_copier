Telegram API megszerzése:
  1. Látogasson el a my.telegram.org oldalra
  2. Jelentkezzen be a telefonszámával
  3. Válassza az "API development tools" menüpontot
  4. Hozzon létre egy új alkalmazást
  5. Másolja ki az API ID-t és API Hash-t
Csatorna ID megszerzése:

A csatorna ID-t megtalálhatja a csatorna információi között, vagy használhatja a @username formátumot is (pl. @mychannel).

Fontos: Az alkalmazás csak olyan csatornákhoz fér hozzá, amelyeknek tagja a bejelentkezett felhasználó.
